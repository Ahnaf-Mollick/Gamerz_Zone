# Gamerz Zone 🎮

**Gamerz Zone** is a modern eCommerce mobile application built with Flutter, designed specifically for selling gaming consoles and accessories. The project focuses on delivering a clean user experience, responsive UI, and scalable architecture for future backend integration.

Most of the core functionalities have already been implemented, making the application nearly production-ready. Simply connect your preferred backend/database service, and the application can be extended into a fully functional eCommerce platform.

## ✨ Features

### 🏠 Home Screen

* Modern and visually appealing UI
* Featured products section
* Category browsing
* Smooth navigation experience

### 🔍 Search System

* Dedicated search screen
* Real-time product searching
* Easy product discovery

### 📂 Category Filtering

* Fully functional category-based filtering
* Dynamic product display based on selected categories

### 📄 Product Details

* Detailed product information page
* Product specifications
* Pricing information
* Quantity selection

### 🛒 Order Management

* Global order management system
* Centralized state handling
*  order data accessible throughout the application

### 📝 Customer Information Collection

* User information form
* Shipping and order details management

### 📋 Order Summary

* Complete order information listing
* Easy review before checkout

### 🗑️ Swipe-to-Delete Functionality

* Slide left to remove products/orders
* Smooth user interaction

### 🎨 Additional Features

* Custom Splash Screen
* Responsive layouts
* Clean UI design
* Reusable widgets
* Scalable project structure
